// What It Does

Not a chatbot.
A runtime.

🏠

Runs Locally

Daemon on port 18800. JSON persistence in ~/.omnis-key/. Air-gap capable. Your data never leaves your machine.

🛡️

HEIMDALL Trust Layer

Every response carries a trust score (0–100), SHA-256 attestation, and a 4-tier verdict: SOVEREIGN / VERIFIED / PROVISIONAL / CONTESTED.

🔭

Causal, Not Correlational

NOVA builds a structural causal model. ECLIPSE reconstructs temporal sequences. You get why, not just what.

🌐

Constellation Packs

POLARIS (gas/compression), SOLSTICE (refinery/logistics), GAIA (food/conflict), AEGIS (defense) — domain intelligence you install like npm packages.

🎙️

JARVIS Voice Mode

Web Speech API for input, ElevenLabs for output. Talk to your runtime. Get COSMIC briefings spoken back to you.

🧠

LLM-Agnostic

Routes through Ollama first, falls back to Claude or GPT. COSMIC engines run regardless — even fully offline if needed.
